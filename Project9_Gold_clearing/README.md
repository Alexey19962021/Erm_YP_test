В проекте было необходимо подготовить прототи модели, прогнозирующей коэффициент восстановления золота из руды. Технологический процесс включает два очень важных этапа: флотацию и очистку. При решении задачи учитывались эффективность обогащения чернового концентрата и финальной очистки в пропорциях (1:3)
